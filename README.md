# IMDb Top Movies Scraper

A Python-based web scraping project that extracts movie information from IMDb Top Rated Movies using Selenium, BeautifulSoup, and Requests and exports the data into CSV files for analysis.

---

## Features

- Scrapes IMDb Top Rated Movies page
- Extracts detailed movie metadata
- Uses Selenium for dynamic page loading
- Uses BeautifulSoup for HTML parsing
- Stores cleaned data in CSV format
- Beginner-friendly Python project
- Useful for data analysis and machine learning datasets

---

## Technologies Used

- Python
- Selenium
- BeautifulSoup4
- Requests
- CSV
- Chrome WebDriver
